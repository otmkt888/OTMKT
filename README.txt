# 阿根廷2026世界盃預測落地頁 - 使用說明

## ⚠️ 重要：檔案重新命名步驟

下載後，請將檔案改名為以下名稱：

### 檔案清單：

1. **index.html** → 保持不變
2. **12bet-logo** → 改名為 `12bet-logo.jpg`
3. **argentina-hero** → 改名為 `argentina-hero.jpg`
4. **second-kv** → 改名為 `second-kv.jpg`
5. **promotion-banner.jpg** → 保持不變

## 📁 正確的檔案結構：

```
你的資料夾/
├── index.html
├── 12bet-logo.jpg
├── argentina-hero.jpg
├── second-kv.jpg
└── promotion-banner.jpg
```

## 🔧 如何改名？

### Windows:
1. 右鍵點擊檔案
2. 選擇「重新命名」
3. 加上 `.jpg` 副檔名

### Mac:
1. 點擊檔案
2. 按 Enter 鍵
3. 加上 `.jpg` 副檔名

## ✅ 完成後：

雙擊 `index.html`，所有圖片應該正常顯示！

## 📞 需要幫助？

如果還有問題，請確認：
- 所有檔案都在同一個資料夾
- 檔案名稱完全一致（包括大小寫）
- 副檔名是 `.jpg`（不是 .jpeg 或 .png）
